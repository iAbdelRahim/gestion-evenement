Detailed Description

Fichier d'en-tête pour les fonctions et structures liées à l'utilisateur. Ce fichier contient les déclarations de fonctions qui gèrent les opérations liées à l'utilisateur, telles que la récupération de films, la réservation de billets, l'affichage de billets et la création de mariages. Il inclut également les en-têtes et les structures nécessaires.
