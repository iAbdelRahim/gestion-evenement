 main.cCeci est le fichier principal pour le programme de gestion d'événements. Ce programme permet aux utilisateurs de se connecter au système, de s'inscrire et d'effectuer diverses actions telles que la création, la modification ou la suppression d'événements. Les administrateurs peuvent se connecter pour accéder au tableau de bord et effectuer des tâches administratives supplémentaires. Le programme se connecte à une base de données MySQL pour stocker et récupérer des données

The function declarations include inUser, insertUser, getUsers, delUser, inEvent, insertEvent, getEvents, updateEvent, delEvent, getEventSessions, insertPlace, insertPlat, inSession, insertSession, delSession, insertTicket, insertFilm, and getFilmsBySessionId.
